1. Опис модуля SCT lilo

1.1. Введення параметрів завантажувача lilo

Доступні наступні параметри для зміни Device boot – дозволяє вказати пристрій куди буде встановлено завантажувач, Pause -- встановлюється час протягом якого буде відображатись список операційних систем. Якщо встановити прапорець disable Timer тоді таймер взагалі не буде відображатись в меню і система буде чекати поки користувач сам не вибере операційну систему із списку. На вкладці Boot menu config можна редагувати саме меню завантаження: добавляти нові операційні системи, видаляти існуючі. Можна добавляти операційні системи двох видів:

  1. 1. Лінукс.

  2. 2. Other (Windows)

При виборі Linux будуть доступні наступні поля для заповнення:

  • Label – вказується ім’я операційної системи під яким вона буде відображатись у меню завантаження.

  • Image – вказується образ ядра Linux.

  • Root – вказується розділ де знаходиться Linux.

  • Аdd -- тут вводиться додаткові параметри lilo, для тих хто знайомий із синтаксисом файлу lilo.conf поле Аdd аналог append.

  • Video – вибір відео режиму для завантаження Linux.

  • Initrd – тут вказується файл initrd* файл ядра.

При виборі Linux будуть доступні наступні поля для заповнення:

  • Label – вказується ім’я операційної системи під яким вона буде відображатись у меню завантаження.

  • Root – вказується розділ де знаходиться операційна система.

Для видалення операційної системи із списку достатньо вибрати її в списку і натиснути кнопку Delete. Кнопка Edit редагує уже існуючий пункт меню. Вибір set default дозволяє встановити ту операційну систему, яка буде завантажуватись по замовчуванню. Після встановлення потрібних параметрів виконання команди Lilo/Set parameters задіє зміни. В новому вікні буде відображатись інформація про застосування параметрів помилки будуть відображатись червоним кольором.

1.2. Встановлення фонового малюнка

Для встановлення фонового малюнка достатньо перейти на вкладку Themes і вибрати малюнок.

1.3. Встановлення власного фонового малюнка

Для тих хто хоче вставити власний фоновий малюнок, необхідно конвертувати перед тим у 8 бітний bmp малюнок із роздільною здатністю 640х480 у якомусь графічному редакторі (наприклад GIMP) і створити файл даних для цього малюнка. Приклад: нехай ви хочете створити фонову завставку myTheme: для цього ви у каталозі themes програми SCT створюєте каталог myTheme у цьому каталозі створюєте свій малюнок із назвою myTheme.bmp і файл myTheme.dat. Приклад файлу myTheme.dat. 

					bitmap=/boot/myTheme.bmp
					bmp-colors=,,53,15,,25
					bmp-table=45p,57p,1,7
					bmp-timer=535p,17p,7,0,5

Як бачите ім’я папки і файлів що у ній, повинні співпадати. А щодо тлумачення цифр звертайтесь у розділ man lilo.conf.
