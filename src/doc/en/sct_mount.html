2.  Description the module of SCT mount

2.1. Short

utilite is graphic building on of command of mount. Setting of the buttons: mount - assembles the chosen section. umount – unmount is chosen section. scan - serves for the exposure of new sections for example USB Flash Disk.

Important
that sections were automatically connected with the parameters preset by you at the load of the system it is necessary to set mode of record in the file of fstab Data/Write to fstab. Before setting this mode information about editing of sections in the file of /etc/fstab is not written down. (It is recommended to create the back-up copy of file of /etc/fstab).

2.2. Work in standart mode

Mount a section it is necessary to choose him and push the button of mount, or avail context dependent menus and the program automatically will define parameters and will connect the chosen section and will show information where the set section was connected. Like carried out a umount section instruction unmount execution. If you need greater control above editing of section commuted in the mode of expert of view\expert. The button of scan is intended for the exposure of new sections which were connected to the computer already after the start of the program. (Example: USB flesh disk and others like that).

2.3. Work in the mode of expert.

In the mode of expert you will be able in hand to set the parameters of editing. This mode befits more experimental users which know naming of hard disks in Linux systems and they know the parameters of connecting of sections. For those the users, who the parameters of editing are unknown accessible system of prompts it is enough to point the cursor of mouse in the window of editing on a parameter which interests you. Editing, unmounting, scanning is conducted like till it is done in the ordinary mode.
