1.  Description of the module of SCT lilo

1.1. Enter parameters lilo loader

Accessible next parameters for the change of device boot – a device allows to specify where a loader will be set, pause -- time is fixed which the list of the operating systems will be represented during. If to set the small flag of disable Timer then a timer in general will not be represented in a menu and the system will expect while an user will not choose the operating system from a list. On the inset of boot menu config it is possible to edit exactly the menu of load: to add the new operating systems, remove existing. It is possible to add the operating systems of two kinds:

  1. 1. Linux.

  2. 2. Other (Windows)

At the choice of Linux there will be the accessible followings fields for filling:

  • Label – the name of the operating system is specified under which she will be represented in the menu of load.

  • Image – appearance of kernel of Linux is specified.

  • Root – a section is specified where Linux is.

  • Add -- here entered additional parameters of lilofor those who acquaintance with the syntax of file of lilo.conf Add weeds the analogue of append.

  • Video is a choice of video of the mode for the load of Linux.

  • Initrd – a file is here specified initrd* file of kernel.

At the choice of Linux there will be the accessible followings fields for filling:

  • Label – the name of the operating system is specified under which she will be represented in the menu of load.

  • Root – a section is specified where the operating system is.

For deleting the operating system from a list it is enough to choose her in a list and push the button of delete. The button of edit edits an already existent menu item. The choice of set default allows to set a tu operating-room system which will be loaded on saying nothing. After establishment of necessary parameters of instruction lilo/set parameters execution will involve changes. In a new window information will be represented about application of parameters of error will be represented a red color.

1.2. Set background image

For establishment of background picture it is enough to pass to the inset of themes and choose a picture.

1.3. Set oters background images

For those who wants to insert an own background picture, it is necessary to convert before tim in 8 bit a bmp picture with settling ability 640х480 in some the editor (for example GIMP) and to create the file of data for this picture. Example: let you want to create background image mytheme: for this purpose you in the catalogue of the SCT/share/lilo/themes program SCT create the catalogue of mytheme in this catalogue create the picture with the name of mytheme.bmp and file of mytheme.dat. Example of file of mytheme.dat. 

					bitmap=/boot/mytheme.bmp
					bmp-colors=,,53,15,,25
					bmp-table=45p,57p,1,7
					bmp-timer=535p,17p,7,0,5

As see the name of folder and files that in her, guilty to coincide. And in relation to interpretation of numbers apply in the section of man lilo.conf.
