1. About programu

The program SCT (System of configuration tools) is a tool of config Linux of the systems. Mainly oriented to work from mylinux 3.1. The program also will work under the management of such os of Linux as Mandrake Linux 10.1, openSuSE, under other os the linux program was not tested. The program SCT uses, both external facilities of adjusting the system and contains the additional modules and programs of adjusting the system. Now accessible: SCT lilo, SCT mount, SCT rpm, SCT DM. These components are set separately they can be free of charge loaded from an official site SKsoft : www.sksoft-gr.narod.ru. About all found out errors and also, if you have a wish of schodo of functional possibilities or program interface ask to report on: sksoft@i.ua.
