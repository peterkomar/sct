Installation and system requirements

For installation of the program start the program of setub and execute pointing which will appear on the screen. As a program will going under your system that is why procedure of installation, depending on your computer, can occupy the perceptible interval of time. If the program is not started then load the program of installation of setub from a cantilever. If you want to look after a process compiling of the program it is necessary to start in a terminal the program of setub –nogui. For installation of the module it is necessary to startsetub_mod.

System requirements.

It is necessary for successful installation of the program and modules:

  1. To have rights for a record in a directory where the program of setub is and where you want to set the program and its components.

  2. Presence in the system of packages of qt and qt-devel.

  3. For the module of SCT rpm a necessity is also a presence of package of rpm-devel.
