1.  Description of the program

The program has two panels. There are groups of the programs and modules in a left panel, in right there are the programs and modules which belong to one group. As all programs are broken to the groups it provides the rapid search of the necessary program. To start the module or program it is enough to point the cursor of mouse on the icon of the program and one time to click the left mouse button. Some programs can be not started – reason of it is absence of these programs in your system. To remove her: check path's to the programs in File/Preferences. Program SCT control center usually the correct will identify your linux and will connect the necessary programs. However if you use linux different from SUSE, Mandrake, varieties of Fedora Core then SCT will not connect the proper programs and you need it will be to do it by manually.

1.1.  Menu File

The menu of file contains two commands preferences and exit.

  • Preferences - is a call of menu of the program settings. It is here possible to specify ways to facilities of adjusting the system, and also to add a program mark for a quick start in a menu or on a desktop.

  • Exit - is a program exit SCT control center.

1.2.  Menu View

The menu of view contains a command:

  • Detailed view - near the icons of the programs the thumb-nail sketch of the program will be represented.
