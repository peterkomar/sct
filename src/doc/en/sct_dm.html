3. Description module of SCT dm

utilite allows to choose manager of registration. Accessible : kdm, gdm, xdm and entered an user. For togo that to set some manager it is enough to choose him and push the button of finish.

Important
programa changes the file of /etc/sysconfig/displaymanager. That is why it is recommended to create the back-up copy of file of /etc/sysconfig/displaymanager.
