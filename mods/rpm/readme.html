Installation and system requirements

For installation of the program start the program setub.sh (or setub_mod.sh for installation of the module ) and execute pointing which will appear on the screen. As a program will building under your system that is why procedure of installation, depending on your computer, can occupy the perceptible interval of time. If window with a license show fails then load the program of installation of setub.sh in a terminal. If you want see compiling it is necessary to start the program of setub –nogui (or setub_mod –nogui) in a terminal.

System requirements.

It is necessary for successful installation of the program and modules:

  1. To have rights for a record in a directory where the program of setub is and where you want to set the program and its components.

  2. Presence in the system of packages of qt3 and qt3-devel.

  3. For the module of SCT rpm a necessity is also a presence of package of rpm-devel.

Інсталяція та системні вимоги

Для інсталяції програми запустіть програму setub.sh (або setub_mod.sh для інсталяції модуля ) і виконуйте вказівки які будуть з’являтись на екрані. Оскільки програма буде збиратись під вашу систему тому процедура інсталяції, залежно від вашого комп’ютера, може зайняти відчутний проміжок часу. Якщо не з'являється вікно із ліцензією тоді завантажте програму інсталяції setub.sh в терміналі. Якщо ви хочете спостерігати за процесом компіляції програми необхідно запустити у терміналі програму setub –nogui (або setub_mod –nogui).

Системні вимоги.

Для успішної інсталяції програми та модулів необхідно:

  1. Мати права запису в директорії де знаходиться програма setub і куди ви хочете встановлювати програму та її компоненти.

  2. Наявність в системі пакетів qt3 та qt3-devel.

  3. Для модуля SCT rpm необхідним є також наявність пакета rpm-devel.
